# 🐦 Twitter Clone – Modern Web Application

A responsive and modern **Twitter Clone** built using **React.js and Vite**, designed to replicate the core user interface and basic functionality of a social media platform. This project demonstrates my ability to build scalable frontend applications using a component-based architecture and modern development tools.

---

## 📖 Overview

The Twitter Clone application provides a clean and interactive user interface where users can view a tweet feed similar to the original Twitter platform. It focuses on frontend design, responsiveness, and performance optimization.

This project helped me understand how real-world social media applications are structured and how to build reusable UI components for scalability.

---

## 🚀 Features

- Responsive and user-friendly interface  
- Tweet feed layout inspired by Twitter  
- Component-based architecture using React.js  
- Fast rendering and optimized performance using Vite  
- Clean UI design with reusable components  
- Mobile-friendly and cross-device compatibility

## ⚙️ Installation & Setup

### Step 1: Clone the Repository
```
git clone https://github.com/marellasiva/twitter-clone.git
```

### Step 2: Navigate to Project Folder
```
cd twitter-clone
```

### Step 3: Install Dependencies
```
npm install
```

### Step 4: Run the Application
```
npm run dev
```

### Step 5: Open in Browser
```
http://localhost:5173
```
